# Ease & Aura — 30-Day Content Flywheel (IG · TikTok · FB)

**Goal of these 30 days:** NOT to make money yet. It's to (1) go live, (2) get reach on video, (3) build the first audience + email leads, (4) learn what content lands. First *sale* is realistically day 45–90 from a cold start — normal, not failure. The 30-day win = a live, consistent presence with growing views and first traffic.

**Effort:** ~20–30 min/day.
**Channels:** Instagram (Reels) + TikTok = primary reach (surface to non-followers, no audience needed). Facebook = secondary (allows clickable links → can drive clicks directly). YouTube = later.
**Link reality:** IG + TikTok have NO clickable links in posts — all traffic goes through **one link in bio**. So set up a **link-in-bio → easeandaura.com quiz** (Linktree or direct). These channels win on reach + time; the sale path is longer than a direct-link channel. Build audience → clicks → sales, in that order.
**Funnel:** Reel/post → "free quiz, link in bio" → easeandaura.com quiz → capsule → shop (Amazon `easeandaura20-20` + Sovrn).

**Who does what:**
- **Claude (me):** designs the carousels/grids (no product photos needed), writes captions + hooks + hashtags, gives Runway prompts for Elle clips, batches content.
- **You:** render Elle clips in Runway, post + repurpose across all 3 channels, daily 5-min engagement, watch which videos pop.
- **Bonus (optional):** each grid also posts to Pinterest as a direct-link pin — free click upside.

---

## Content types (all repurpose across IG + TikTok + FB)
1. **Elle Reels** — short motion clips (the 1 you're rendering; more later). Hook → capsule tip → "free quiz, link in bio."
2. **Carousels** — the capsule grids + tip graphics I build (60-30-10, "5 outfits / 3 pieces," "shop at any budget"). Save-worthy, swipeable.
3. **Talking-tip captions** — Elle still + a strong text hook ("The 3 pieces every neutral capsule needs").

**Day 1 — Foundation**
- [ ] Set up **link-in-bio** (Linktree/direct) → the quiz (with `?utm_source=instagram` etc.)

- [ ] TikTok: account, same profile pic + bio + link
- [ ] FB: upload profile + cover, post + PIN launch Post 1 (from `first-posts.md`)

**Day 2–7 — First content**
- [ ] Post the **Elle intro Reel** to IG + TikTok + FB (once you've rendered it)
- [ ] Post the **"Neutral Capsule" carousel** to IG + FB (I deliver it)
- [ ] Aim **3–4 posts this week** total, cross-posted to all 3
- [ ] 5 min/day: reply to every comment, follow/engage 5 capsule-style accounts

**Week 1 success =** all 3 profiles live, link-in-bio working, 3–4 posts up, first views. No sales expected.

## WEEK 2 — Find what lands (Days 8–14)
- [ ] I deliver 3 new carousels (seasonal capsule, "shop at any budget" toggle, work capsule) + 2 Elle Reel prompts
- [ ] Render 1–2 more Elle clips in Runway
- [ ] **4 posts**, cross-posted; vary the hook each time
- [ ] Watch which video gets the most views/saves — that's your signal

**Week 2 success =** ~8 posts live, one starting to out-perform, first followers, first link-in-bio clicks (trickle).

---

## WEEK 3 — Double down (Days 15–21)
- [ ] Take the best-performing post from Wk 1–2 → I make 3 more *like it* (same hook style/format)
- [ ] **4 posts**, cross-posted
- [ ] First email to any quiz leads (welcome + their capsule + soft shop nudge)

**Week 3 success =** a post or two with real reach, followers growing, tens of link clicks, first quiz completions + email leads. A sale is a bonus.

---

## WEEK 4 — Read the data (Days 22–30)
- [ ] **4 posts**, keep the winning format
- [ ] Day 30 review (I pull it together): views, follows, saves, link-in-bio clicks, quiz starts/completions, email leads, any sales
- [ ] Decide next 30: scale the format that worked; if flat, change the hook/format (not the plan)

**Day 30 success =** live on 3 channels, ~15 posts, a proven content format, first audience + email leads + traffic. That's what the first sale grows from.

---

## The one rule
**Consistency + repurposing.** Make once, post to all 3. 4 posts/week for 4 weeks with one that lands > a burst then silence. The algorithm rewards steady, and one Reel hitting can change everything — but only if you keep feeding it.

## Realistic ladder (cold start — be kind to yourself)
| | Views | Followers | Link clicks | Sales |
|---|---|---|---|---|
| Wk 1 | first views | handful | ~0 | 0 |
| Wk 2 | hundreds–few k | trickle | first few | 0 |
| Wk 3 | one post may pop | growing | tens | bonus |
| Wk 4 | compounding | growing | tens+ | maybe first |

First reliable sales: **day 45–90.** IG/TikTok build audience first — clicks and sales follow the reach. Keep going past day 30; that's when it usually turns.
